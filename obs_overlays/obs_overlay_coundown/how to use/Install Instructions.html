OBS overlay - Countdown

About this overlay

This overlay is showing a countdown on top of the stream.

The overlay source is a web page format which includes the markup code and editable text

Install and use process is not complex and does not require external resources because all files are inside downloaded folder.

Install instructions

How to use this overlay and how to edit it?

In order to edit the content of the Countdown.html you must edit the source of the web page. Notepad++ (Windows users) or TextWrangler (MAC users) will do the job.

To use this overlay on your stream follow the install steps suggested below

  1. Open OBS
  2. From Sources zone click the +
  3. Next step is to select Browser from available options [screenshot]
  4. Create New will show up, give it a name, easy to spot [screenshot]
  5. Local File - Browse. Browse for the file Countdown.html. Make sure you select Local File option [screenshot]
    Settings visible in screenshot
  6. Drag the red corners to fit your main screen source for better position [screenshot]
  7. Optional, you can play with Transition and Fade Transition options when triggering this overlay, it is up to you

Editing the source code

How to set the timer's countdown

Open the html page using Notepad++ or TexWrangler (or any other WYSIWYG editor) and search for this particular piece of code:

<div id="hms" class="cl-12 tcenter orange">00:10:15</div>

Change the digits manually
00:15:22 - will start the contdown 0 hours 15 minutes and 22 seconds
01:25:32 - will start the contdown 1 hour 25 minutes and 32 seconds
00:05:12 - will start the contdown 0 hours 5 minutes and 12 seconds
Default is 00:10:15 - 0 hours, 10 minutes and 15 second contdown
DIGITS ONLY, no additional spaces, text or other characters

Style-it-up

Custom classes can be added in order to change the color and position of the countdown timer

Edit the color using the class values:

<div id="hms" class="cl-12 tcenter orange">00:10:15</div>

change the orange class with green or blue to make the digits green or blue

Optional. You can also reposition the contdown using another set of classes

Default is set to top: <div id="clock" class="cl-12 timer no top">

Set to bottom: <div id="clock" class="cl-12 timer no bottom">

Set to middle: <div id="clock" class="cl-12 timer no middle">

Skip this suggestion if you like because you can reposition the overlay by dragging it on OBS

Edit the text visible at the end of the countdown

When the contdown ends a custom text shows up. You can edit this text from the source

document.getElementById('clock').innerHTML='Thank You.... I will be online as soon as possible';

Edit only the plain text between single quotes
