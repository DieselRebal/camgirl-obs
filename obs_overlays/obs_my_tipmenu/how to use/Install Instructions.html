OBS overlay - My Tip Menu

About this overlay

This overlay is showing animated graphics on top of the video stream.

The overlay source is a web page format which includes the markup code, graphic resources and editable text

Install and use process is not complex and does not require external resources because all files are inside downloaded folder.

Install instructions and Editing suggestions

Skip some of the steps or suggestions if don't apply when you are using the Source Editor

How to use this overlay and how to edit it?

In order to edit the content of the MyTipMenu.html you must edit the source of the web page. Notepad++ (Windows users) or TextWrangler (MAC users) will do the job.

To use this overlay on your stream follow the install steps suggested below

  1. Open OBS
  2. From Sources zone click the +
  3. Next step is to select Browser from available options [screenshot]
  4. Create New will show up, give it a name, easy to spot [screenshot]
  5. Local File - Browse. Browse for the file MyTipMenu.html. Make sure you select Local File option [screenshot]
    Settings visible in screenshot
  6. Drag the red corners to fit your main screen source for better position [screenshot]
  7. Optional, you can play with Transition and Fade Transition options (OBS Studio Mode must be active) when triggering this overlay, it is up to you

Editing the source code

As mentioned above, the overlay can be edited using any HTML/CSS editor (WYSIWYG) like Notepad++, TextWrangler or provided Source Editor (no install necessary)

Open the html file in your favorite editor and alter the markup codes, swap images, edit text ...
